## Installation

There is no need to install anything.

[Visit the website here.](https://willfelder.github.io/netflix-homepage/)

If you prefer:

- Clone repository

```
git clone https://github.com/willfelder/netflix-homepage.git
``` 
- Use your favorite IDE
- Execute npm install
- Execute npm start
